# :books: Tutoriais

## :snake: [Criando o venv (Virtual Environment) do Python](/pages/tutoriais/criando-venv-python)

Criar um ambiente virtual (venv) no Python é uma prática essencial para gerenciar dependências de projetos de forma isolada. Isso permite que você mantenha suas bibliotecas organizadas e evita conflitos entre diferentes projetos.

## :bar_chart: [Evitando valores nulos com Coalesce no SQL](/pages/tutoriais/select-coalesce-sap.md)

Exploramos a função COALESCE no SQL, uma ferramenta essencial para lidar com valores nulos em tabelas de dados. É demonstrado como essa função pode ser utilizada para garantir que sempre haja um valor retornado, mesmo quando alguns campos estão vazios, inclusive em ambientes SAP.
